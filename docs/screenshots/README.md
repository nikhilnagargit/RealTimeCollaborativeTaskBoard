## üé® Screenshot Guidelines

### Capture Settings
- **Browser**: Chrome or Firefox (latest)
- **Window Size**: 1920x1080 (Full HD)
- **Zoom Level**: 100%
- **Device**: Desktop view

### Quality Standards
- **Format**: PNG (for crisp UI elements)
- **Compression**: Optimize with tools like TinyPNG
- **File Size**: Keep under 500KB per image
- **Naming**: Use lowercase with hyphens (e.g., `task-modal.png`)

### Content Guidelines
- **Clean Data**: Use realistic but clean sample data
- **No Personal Info**: Don't include real names or sensitive data
- **Consistent Theme**: Use the same color scheme across screenshots
- **Highlight Features**: Make sure key features are visible
